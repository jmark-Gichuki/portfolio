üîç Brute Force Attack Simulation on DVWA Using Hydra

Tools: Hydra, DVWA (Damn Vulnerable Web Application), Kali Linux, Burp Suite / Wireshark, curl | 2025

Developed an ethical, lab-based brute-force simulation using Hydra against a locally hosted DVWA instance to evaluate authentication weaknesses. Configured Hydra with a controlled wordlist and a test account to automate login attempts, captured and analyzed server responses, and validated successful authentication in a safe environment. The project produced clear evidence of how weak or common passwords and missing protections (rate-limiting, account lockout, MFA) enable automated attacks, and it informed practical mitigation recommendations. This work strengthened my skills in offensive/defensive testing methodology, automated tooling, log analysis, and security-hardening strategies.
